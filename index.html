I'll tell U what I learned about PHP

Pada hari pertama, sebagai tahap awal saya belajar PHP, saya hanya mempelajari dan mencoba memahami tentang pengertian dari PHP itu sendiri.
Jadi, PHP adalah salah satu jenis bahasa pemrograman yang sering digunakan untuk pembuatan dan pengembangan situs web yang bisa digunakan bersamaan dengan HTML.
PHP itu sendiri adalah singkatan dari Hypertext Preprocessor, yang merupakan bagian dari bahasa skrip seperti JavaScript dan Phyton.
PHP ini hampir mirip dengan JS dan Phyton, bedanya adalah PHP sering digunakan untuk komunikasi sisi server, sedangkan JS bisa untuk frontend dan backend
sementara phyton hanya untuk backend atau sisi server.
